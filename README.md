# Overview
Toy Project - Background(가칭)은 개인 독서다이어리 서비스다.
게시판, 로그인 기능 및 서적 검색, 리뷰 기능을 포함한다.

## 주요기능

### 구현예정
+ 회원관리 기능 
    + 회원가입 
        + 아이디 중복 검사 
        + 아이디/비밀번호 입력제약 
        + 개인정보 암호화 저장
    + 약관 및 개인정보수집 동의 
    + 로그인, 로그아웃 
    + 아이디/비밀번호 찾기 
    + 회원정보 관리(목록보기, 회원정보보기, 변경, 탈퇴)
    

+ 게시판 기능 
    + 페이징 처리 
    + 댓글 순서 관리 
    + 대댓글(답글)
    + 목록보기, 게시글 쓰기, 게시글 수정, 게시글 삭제, 게시글 검색
    + 스마트 에디터 사용 
    + 첨부파일 저장 
    + 알림기능 (like, comment, ...)
    + 비밀글 기능 
    + 게시판 종류 및 다른점 처리 (방문록, Q&A, 공지사항, 게시판, 자료실, FAQ) 
    + 게시판 권한관리(읽기, 쓰기, 답변, 첨부 허용여부 관리) 
    + 다중 게시판 만들기(게시판이 여러 개일 때 관리방법) 
    + 게시판 관리(목록보기, 추가, 변경, 삭제)

+ 서적 관련
    + 서적 검색
        + 검색시 어시스턴트 기능 
    + 서적별 평점 기능
    + 좋아요, 별점
    + wish 
    + 읽은 서적 분야별 차트로 보여주기.
+ SNS 연동 기능 
    + 소셜 로그인, 게시글 공유(카톡, 페이스북, 트위터)
    

+ 반응형 웹 만들기  
    + 휴대폰 to PC 접속할 때 동작
